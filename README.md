# Virtual Psychiatrist Interviewer

This project is an AI-based virtual psychiatrist interviewer that analyzes user responses using Natural Language Processing (NLP) and Machine Learning.

## Features
- Simulates a psychiatrist-style interview
- Analyzes mental health responses
- Uses NLP techniques for text processing
- Machine learning model for prediction

## Technologies Used
- Python
- NLP
- Machine Learning
- Google Colab

## Project Structure
- backend : Application backend code
- report : Project documentation
- ppt : Project presentation
